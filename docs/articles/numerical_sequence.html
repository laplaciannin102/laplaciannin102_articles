  • 数が列になったもの.

定義

以下の様な, 自然数を定義域, 実数または複素数を値域とする写像を数列と定義する.


$$ \begin{aligned} f: \mathbb{N} \rightarrow \mathbb{R} \\ \\ \text{または} \\ \\ f: \mathbb{N} \rightarrow \mathbb{C} \\ \end{aligned} $$



例

等差数列

  • 1, 2, 3, 4, 5, ⋯
    • f : n ↦ n

フィボナッチ数列

  • 0, 1, 1, 2, 3, 5, 8, 13, ⋯
    • f(1) = 0
    • f(2) = 1
    • f(n + 2) = f(n) + f(n + 1) (n ∈ ℕ)
